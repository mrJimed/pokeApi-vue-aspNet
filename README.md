# PokeApi (Возможности)
* Получение данных с https://pokeapi.co/
* Фильтрация данных по имени покемона
* Отображение данных (имя, hp и сила атаки)
* Пагинация с возможностью выбора кол-ва отображаемых покемонов на странице
* Аутентификация и регистрация
* Кэширование полученных покемонов 
* Бои между покемонами (одного бойца выбирает пользователь, второй выбирается случайно)

# Frontend
* Vue 3
* Vue Router
* AutoAnimate
* lodash.debounce
* Tailwind
* Axios

# Backend
