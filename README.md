# PokeApi (Возможности)
* Получение данных с https://pokeapi.co/
* Фильтрация данных по имени покемона
* Отображение данных (имя, hp и сила атаки)
* Пагинация с возможностью выбора кол-ва отображаемых покемонов на странице

# Frontend
* Vue 3
* lodash.debounce
* Tailwind

# Backend
* C# .NET 8
* ASP NET Core MVC
* Newtonsoft.Json
